# Website Mirror

Vite + React + TypeScript starter and website-analysis scaffold using TailwindCSS and Radix UI. Built for rapid prototyping and learning modern frontend patterns.

---

## Features

- Vite dev server for fast development
- TypeScript + React 19
- TailwindCSS utility styling
- Radix UI primitives and accessible components
- Preconfigured linting (ESLint) and formatting (Prettier)

---

## Technologies used

- TypeScript
- React 19
- Vite
- TailwindCSS
- Radix UI
- ESLint & Prettier

---

## Quick start

1. Clone:

```bash
git clone https://github.com/DEEPAKS110/website-mirror.git
cd website-mirror
```

2. Install:

```bash
npm install
```

3. Run dev server:

```bash
npm run dev
```

4. Build:

```bash
npm run build
```
